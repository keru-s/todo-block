# Triage labels

| Role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Waiting for initial evaluation |
| `needs-info` | `needs-info` | Waiting for more information |
| `ready-for-agent` | `ready-for-agent` | Clear enough for an agent to complete |
| `ready-for-human` | `ready-for-human` | Requires human work or judgment |
| `wontfix` | `wontfix` | Will not be pursued |

Engineering skills should use the GitHub label mapped to each role.
